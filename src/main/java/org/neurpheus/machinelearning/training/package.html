Organizes training sets.

Package Specification

The {@link org.neurpheus.classification.training.TrainingSet} interface represents a set of training examples usead by a learning process. The {@link org.neurpheus.classification.training.TrainingExample} interface represents a single training example which is managed by the training set.

Following implementations of the traing set interface are available:

  • {@link org.neurpheus.classification.training.MemoryTrainingSet} - training set which stores all examples in a RAM.
  • {@link org.neurpheus.classification.training.FileTrainingSet} - training set which stores a large set of training examples in a file.
